### Configure

Install dependencies

```bash
npm install --frozen-lockfile
```

Run the project

```bash
npm run dev
```

### Installed libs

- [axios](https://axios-http.com/)

- [lucide-react](https://lucide.dev/)
- [zod](https://zod.dev/)
